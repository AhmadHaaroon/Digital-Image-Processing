This project involves several image processing methods, including:
  - Edge detection:
    . Marr Hildreth Edge Detector
    . Canny Edge Detector
  - Image Enhancement/Alterations:
    . Negative Image
    . Thresholding
  - Image Resizing:
    . Nearest Neighbour
    . Bilinear

Several processing filters, such as guassian, laplacian, sobel etc are also implemented in this project.
